Why EnteleWALLET Lite is safer

EnteleWALLET Lite is designed for verification without taking custody.

  • No seed phrases or private keys are stored.
  • EnteleWALLET does not take custody of funds.
  • It connects only to existing wallets.
  • Verification uses safe message signatures.
  • Users remain in control of approvals and transactions.

Supported wallet routes

Supported routes include MetaMask, Rabby, WalletConnect wallets, Coinbase Wallet, Trust Wallet, Rainbow, and Ledger through supported integrations.

MetaMask Rabby WalletConnect wallets Coinbase Wallet Trust Wallet Rainbow Ledger

Message signature safety

A valid signature only verifies that you control a wallet address.

  • It does not transfer tokens.
  • It does not approve spending.
  • It does not create a blockchain transaction.
  • It does not give custody of assets.
  • It does not require gas.

Never sign messages you do not understand.

What we will never ask for

EnteleWALLET, EnteleKRON, TVK Group, TVK Labs, SOVRA, support staff, admins, partners and community moderators will never ask for the following:

  • Seed phrase
  • Private key
  • Recovery phrase
  • Wallet password
  • Remote access to your computer
  • Payment through social media messages

Phishing prevention

Use these rules before every wallet connection or signature request.

  • Never trust payment addresses sent through Telegram, Discord, X, Instagram or email.
  • Always verify payment instructions inside the official investor dashboard.
  • Never install fake wallet extensions.
  • Bookmark the official domains.
  • Verify SSL and HTTPS before wallet access.
  • Avoid sponsored search-result links for wallet access.
