Signature Safety

A valid EnteleWALLET signature request should only prove wallet ownership and account control.

✓

It must only verify wallet ownership.

✓

It must not transfer tokens.

✓

It must not approve spending.

✓

It must not create a blockchain transaction.

✓

It must not cost gas.

Never sign messages you do not understand. Pause and verify the message content before approving any signature request.
